使用鼠标事件实现拖拽

三个步骤

  1. 当点击要拖拽元素时（mousedown），获取元素当前位置。
  2. 拖拽时（mousemove），改变元素的位置。
  3. 释放鼠标时（mouseup），停止拖拽

定义dragElement方法

实现细节

HTML drag and drop uses the DOM event model and drag events inherit from mouse events.

A typical drag operation:
begins when a user selects a draggable element with a mouse,
moves the mouse to a droppable element
and then releases the mouse

DataTransfer object
the type of drag
the drag’s data
the type of each drag item
Each DataTransfer object contains an items property which is a list of DataTransferItem objects
represents a single drag item
has a kind property、a type property

  2. Define the drag’s data

  4. Define the drag effect
  5. Define a drop zone
