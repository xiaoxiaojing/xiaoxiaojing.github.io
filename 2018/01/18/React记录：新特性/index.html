React记录：新特性

Posted on 2018-01-18 | Post modified: 2018-01-18 | In REACT

React v16.2.0

React.Fragment

可以使用React.Fragment来包裹多个React Element。渲染后不会创建多余的DOM

React.Fragment有且仅有一个属性key

JSX也实现了对应的功能，但是不能设置key属性

React v16.0

支持新的返回类型：fragments 和 strings

在render()方法中，return的值可以是一个数组。但是每个React Element都必须添加属性key。

在render()方法中，return的值也可以是一个string

使用error boundaries处理错误

定义：定义了周期函数componentDidCatch的组件就是一个error boundaries
特点：error boundaries在发生错误的时候，能捕获错误，并在它的subtree上渲染出fallback UI
需要主要的地方：

  • Only class components can be error boundaries；
  • 可以定义一个error boundaries，并重复使用
  • error boundaries只会捕获他子树的 生命周期函数 产生的错误
