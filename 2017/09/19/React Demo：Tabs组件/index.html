React Demo：Tabs组件

Posted on 2017-09-19 | Post modified: 2018-06-23 | In React系列

效果展示



设计思路

  1. react的思想是数据驱动，所以将Tabs抽象成一份数据

  2. 数据最终渲染成的DOM结构

  3. 组件设计

    • 父组件Tabs，用来渲染Tabs的结构

    • 子组件Tab，用来渲染Tabs的nav部分

    • 子组件TabContent，用来渲染Tabs的content部分

组件实现

  1. 子组件Tab和TabContent很容易编写，就是一个纯函数（只要注意控制标签的隐藏和显示）

  2. 父组件Tabs的实现

    • 方法1：将数据tabs_data传递给父组件Tabs，然后对数据进行处理，构建出子组件。（不直观，如果数据不是需要的格式，还要进行一次数据构造）

    • 方法2：在父组件使用阶段，将数据tabs_data分解后传送给父组件。（直观，符合React的设计思想）

  3. 父组件的实现

    • 通过Children值构造nav

    • 通过Children值构造content

    • 组装
